Schubert in 1827 (oil on canvas, by Anton Depauly)

Franz Schubert's final chamber work, the String Quintet in C major (D. 956, Op. posth. 163) is sometimes called the "Cello Quintet" because it is scored for a standard string quartet plus an extra cello instead of the extra viola which is more usual in conventional string quintets.
